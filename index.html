System Information

Select the correct CCT.DAT.inf file based on your SMEG system:

  • SMEG systems: /SMEG_UPG/DATA/CCT.DAT.inf
  • SMEG + systems: /SMEG_PLUS_UPG/DATA/CCT.DAT.inf
  • SMEG iv2 systems: /SMEG_IV2_UPG/DATA/CCT.DAT.inf
  • RT6 systems: /CCT.DAT.inf (use the ROOT directory file)
Recommendation

Generate all 4 possible codes and write them down. Test them in your car until you find the correct one to avoid multiple trips.
